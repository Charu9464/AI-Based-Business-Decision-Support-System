# 🛒 AI-Based Business Decision Support System

## 📌 Project Overview

This project is an AI-Based Business Decision Support System developed using the Blinkit Sales Dataset. The goal of this project is to analyze retail sales data, build an interactive Power BI dashboard, generate business insights, and automate customer communication using Zapier.

The project demonstrates how data analytics and AI-assisted insights can support business decision-making by identifying sales trends, customer behavior, and product performance.

---

## 🎯 Objectives

- Analyze Blinkit retail sales data.
- Clean and preprocess the dataset using Excel.
- Build an interactive Power BI dashboard.
- Generate business insights using AI.
- Design a promotional poster in Canva.
- Automate email delivery using Zapier.

---

## 📈 Key Business Insights

- Supermarket Type 1 generated the highest sales.
- Regular fat products contributed more revenue than low-fat products.
- Tier 3 locations recorded strong overall sales performance.
- Fruits, vegetables, and snack foods were among the top-selling categories.
- Medium-sized outlets generated higher revenue compared to small outlets.
- Customer ratings remained consistently positive across most product categories.
